OptionalautoDimensions

autoDimensions?: boolean

automatically calculate dimensions from the available attributes of both the local SVG element (on which action is used) and the remote SVG

For example, if you specify only width to the local SVG element, the height will automatically be calculated from the remote SVG

For this to work, width & height must be "extractable" from the remote element, that is, the remote SVG must either have the viewBox or both width and height attributes that is in the same unit.
