Type Alias OptionsData

Defines the common dialog configuration data.

type OptionsData = {
    alwaysOnTop?: boolean;
    buttons?: { [key: string]: TJSDialog.ButtonData };
    content?: Embed | string;
    default?: string;
    draggable?: boolean;
    focusAuto?: boolean;
    focusFirst?: boolean;
    focusKeep?: boolean;
    focusTrap?: boolean;
    minimizable?: boolean;
    modal?: boolean;
    modalOptions?: OptionsModal;
    notifyError?: boolean;
    onClose?: string | ((data?: { application: TJSDialog }) => any);
    rejectClose?: boolean;
    resizable?: boolean;
    resolveId?: boolean;
    title?: string;
    transition?: OptionsTransition;
    zIndex?: number | null;
}

true

Optionalmodal

false

Additional options for modal dialog display.

OptionalnotifyError

notifyError?: boolean

When true and an error is thrown in dialog callback functions post a UI error notification.

Default Value

false

OptionalonClose

onClose?: string | ((data?: { application: TJSDialog }) => any)

Callback invoked when the dialog is closed; no button option selected. When defined as a string, any matching function by name exported from the content Svelte component is invoked.

OptionalrejectClose

rejectClose?: boolean

When true and a Promise has been created by TJSDialog.wait and the Promise is not in the process of being resolved or rejected on close of the dialog any onClose function is invoked and any result that is undefined will cause the Promise to then be rejected.

Default Value

false

Optionalresizable

resizable?: boolean

When true, the dialog is resizable.

Default Value

false

OptionalresolveId

resolveId?: boolean

When true and resolving any Promises and there are undefined results from any of the button callbacks, the button ID is resolved.

Default Value

false

Optionaltitle

title?: string

The dialog window title.

Optionaltransition

transition?: OptionsTransition

Transition options for the dialog.

OptionalzIndex

zIndex?: number | null

A specific z-index for the dialog.
