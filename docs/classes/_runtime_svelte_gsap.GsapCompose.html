Class GsapCompose

Provides a data driven ways to connect a TJSPosition instance with a GSAP timeline and tweens.

GsapPosition.timeline supports the following types: 'add', 'addLabel', 'addPause', 'call', 'from', 'fromTo', 'set', 'to'.
