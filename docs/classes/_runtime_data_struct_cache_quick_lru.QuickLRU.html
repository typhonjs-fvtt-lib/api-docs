constructor

  • new QuickLRU<KeyType, ValueType>(options): QuickLRU<KeyType, ValueType>

  • Simple "Least Recently Used" (LRU) cache.

    The instance is an Iterable of [key, value] pairs so you can use it directly in a for…of loop.

    Type Parameters

    • KeyType

    • ValueType

    Parameters

    • options: Options<KeyType, ValueType>

    Returns QuickLRU<KeyType, ValueType>

    Example

    import { QuickLRU } from '#runtime/data/struct/cache/quick-lru';

    const lru = new QuickLRU({maxSize: 1000});

    lru.set('🦄', '🌈');

    lru.has('🦄');
    //=> true

    lru.get('🦄');
    //=> '🌈'

    Overrides Map.constructor
